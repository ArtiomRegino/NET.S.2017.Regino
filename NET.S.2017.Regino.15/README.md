# NET.S.2017.Regino.15

Постановка задачи. 
Товар характеризуется категорией (одно из предопределенных значений), названием, описанием, ценой и изображением. Покупка – это набор разных товаров, для каждого товара в покупке задано его количество.

1.	Необходимо создать базу данных для хранения указанных сущностей (Entity Framework Database First, Entity Framework Code First to an Existing Database). 
2.	Необходимо создать набор классов для представления указанных сущностей (Entity Framework Code First to a New Database). 
3.	Реализовать классы-сервисы для управления сущностями (добавления, удаления, изменения) в базе данных. Предусмотреть импорт информации в XML-файл, а также экспорт из XML-файла в базу данных (опционально).
4.	Протестировать работу классов с использованием консольного приложения.
